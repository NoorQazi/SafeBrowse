# SafeBrowse

## About
This project is a Chrome extension made to improve safe browsing. It helps users check if a website is secure and also shows how strong their password is.

## Features
- Checks password strength (weak, medium, strong)
- Detects whether a website is secure (HTTPS or not)
- Gives simple feedback to the user

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)

## How to Run

1. Run the backend:
   python app.py

2. Open Chrome and go to:
   chrome://extensions/

3. Turn on Developer Mode

4. Click on "Load unpacked"

5. Select the project folder

## Working
First, the user enters a password and the extension checks its strength.  
Then when the user clicks on "Check Website", it checks if the website is secure and shows the result.

## Output
- Password strength: Weak / Medium / Strong
- Website: Secure / Not Secure
